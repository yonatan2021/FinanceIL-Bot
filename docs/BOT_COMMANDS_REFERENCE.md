# 🤖 טלגרם בוט — הפניה מלאה

---

## הפקודות הראשיות

### `/start` או `/menu`
הצג את התפריט הראשי עם כל האפשרויות.

**Payload**: אין  
**Response**: תפריט אינטראקטיבי עם כפתורים

---

## התפריט הראשי

ברגע שאתה שולח `/start`, מופיע תפריט עם כפתורים:

| כפתור | פעולה | תוכן |
|-------|-------|-------|
| **💰 יתרות חשבונות** | `menu:balances` | מציג יתרות כל חשבון מכל בנק |
| **📊 סיכום חודשי** | `menu:summary` | הוצאות החודש לפי קטגוריה |
| **📋 עסקאות אחרונות** | `menu:transactions` | 10 עסקאות אחרונות (עם עימודים) |
| **📈 תקציב החודש** | `menu:budget` | התקציב החודשי + התראות |
| **🔧 ניהול** | `menu:settings` | (ניתן רק לאדמין) |
| **◀️ חזור** | `menu:back` | חזור לתפריט הראשי |

## תפריט ניהול (Admin Only)

### 🔧 ניהול (`menu:settings`)

כפתורים זמינים רק לאדמין:

| כפתור | פעולה | תוכן |
|-------|-------|-------|
| **🔄 הריץ סקרייפר** | `admin:scraper` | מפעיל סקריפט סקריפ בנקים |
| **👥 משתמשים** | `admin:users` | רשימת כל המשתמשים המורשים |
| **📋 לוג אחרון** | `admin:logs` | תוצאה של הריצה האחרונה של הסקרייפר |
| **◀️ חזור** | `menu:back` | חזור לתפריט ראשי |

---

### 🔄 הריצ סקרייפר (`admin:scraper`)

מפעיל מנוע סקריפ לעדכון עסקאות מהבנקים.

**שליחה:**
```
POST /api/scrape
Header: x-internal-secret: [INTERNAL_API_SECRET]
```

**תגובה:**
- ✅ הסקרייפר הושלם בהצלחה.
- ❌ שגיאה: סטטוס [status]
- ❌ שגיאת חיבור לשרת.

---

### 👥 משתמשים (`admin:users`)

רשימה של כל המשתמשים + תפקידם.

**הצגה:**
```
משתמשים

Yonatan (123456789) | role: admin | פעיל ✅
Friend1 (987654321) | role: viewer | פעיל ✅
Friend2 (111222333) | role: viewer | לא פעיל ❌
```

---

### 📋 לוג אחרון (`admin:logs`)

מידע על הריצה האחרונה של הסקרייפר.

**הצגה:**
```
לוג אחרון

התחיל: 17.04.26 09:15
סיים: 17.04.26 09:27
עסקאות שהובאו: 42
סטטוס: success ✅
```

## הצפנה וביטחון

### מי רואה מה?

| תפקיד | גישה |
|-------|------|
| **admin** | הכל (יתרות, סיכום, עסקאות, תקציב, ניהול) |
| **viewer** | יתרות, סיכום, עסקאות, תקציב בלבד |

ניסיון לגשת ל-`menu:settings` כ-viewer:
```
❌ אין לך הרשאה לפאנל זה.
```

## טיפול שגיאות

### בוט לא משיב

**בדוק:**
1. `TELEGRAM_BOT_TOKEN` בקובץ `.env.local`?
2. Token תקף? (`curl https://api.telegram.org/bot{TOKEN}/getMe`)
3. סקריפט בוט רץ? (`npm run dev` בקונסול)

**תיקון:**
```bash
pkill -f "node.*bot"  # הרוג את הבוט
npm run dev           # התחל מחדש
```
