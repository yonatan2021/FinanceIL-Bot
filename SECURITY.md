# Security Policy

## Supported Versions
Only the latest version of this software is currently supported.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please report it responsibly.

**DO NOT** create a public GitHub issue for security vulnerabilities.

Instead, please report any security issues to the project maintainer via a private channel if possible, or wait until a private reporting feature is enabled on this repository.

### What NOT to include
- No API keys
- No Telegram bot tokens
- No production database URLs
- No encryption keys
- No personal financial data
